Keyboard Help

The following short cut keys are available when the focus is in the message list.

Key Description
s Compose a new message.
c Compose a reply to the selected message.
f Set focus to the message search window.
g Go to a message.
h Go to the next unread priority message.
o Go to the message to which the selected message is a reply.
i Mark the selected message and all replies ignored. If multiple messages are selected, only those messages are marked ignored.
p Mark the selected message and all replies priority. If multiple messages are selected, only those messages are marked priority.
r Mark the selected messages read.
m Flag the selected message.
z Mark the thread starting from the selected message as read then advances to the next unread message if there is one.
Z As 'z' above but goes to the root message of the next unread thread.
Enter Mark the selected message read then find the next unread message in any folder.
/ Mark the selected message read then find the next unread message in any folder and move to its root message.
Space Scroll down through a multi-page message. (Note: Vole/Vinkix/Vienna versions before 1.6.4 also went to the next unread message at the end of the page.)
Left Go to the root message of the current thread.
Right Go to the root message of the next thread.
Backspace Go to the previously viewed message.
Del Delete the selected messages.
