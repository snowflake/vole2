Credits

Vienna was written by Steve Palmer (spalmer@cix.co.uk).

Additional code by Christine Caulfield (tykepenguin@cix.co.uk -- this address is now longer active).

From 2010, Dave Evans (devans@cix.co.uk) took over maintenance. In December 2011 he renamed Vienna to Vinkix. In July and August 2012 Vinkix was renamed to Vole (possible acronym for Vienna Off Line Engine and species of small, cute, furry mammals)

Vienna/Vinkix/Vole is built using the following technologies:

  • Cocoa Framework using Objective-C.
  • SQLite public domain SQL database (http://www.sqlite.org).
  • MacPAD (http://macshareware.net/sdk.html) for the update technology.
  • Blackhole Media SQLDatabase 0.1 Objective-C wrapper for SQLite (http://blackholemedia.com/code/).
  • Objective-C wrapper for the Apple XML parser (http://iharder.sourceforge.net/macosx/xmltree/)
  • RSS Parser from Brent Simmons (http:///)
  • lrzsz Zmodem programs <uwe-lrzsz@bulkmail.ohse.de> taken from Darwin ports http://darwinports.opendarwin.org/

Special thanks are due to Jim Strong and Sam Aouane for their help in testing Vienna. Thanks are also due to Bryan Ford, Richard Buckle, Christine Caulfield, Paul Wakeford, Nigel Martin, Jon Honeyball, D.J.Walker-Morgan, Ian Moran, Rod Mould, Rick Andrews, Adam Wood and Jane Palmer.
