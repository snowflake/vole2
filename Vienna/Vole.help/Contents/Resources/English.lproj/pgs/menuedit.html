Edit menu


  • Copy URL

    Copy URL copies a Cix Hotlink for the current message to your Clipboard. When you compose a new mesage you can paste the Hotlink into it.

  • Special Characters

    Special Characters (or Emoji and Symbols in some versions os OS X) brings up the Character Viewer window. In Character Viewer you can copy and paste characters that are not available from your Keyboard.

    Important: Please read Character Viewer before using it.
