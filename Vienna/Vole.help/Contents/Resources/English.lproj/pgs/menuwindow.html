Window menu

  • Activity Window

    The Activity Window displays the interaction with Cix during a blink.

  • Tasks Window

    Displays the tasks which will be performed on the next blink. It also enables you to cancel tasks that you do not want to run.

    Note: Messages that are pending to be posted to Cix are stored in Vole's Out Basket folder. This is near the top of Vole's list of folders. You should delete a message from the Out Basket if you no longer wish it to be posted.

  • Browser

    This displays a list of Cix conferences togther with descriptions and the date a message was last posted.
